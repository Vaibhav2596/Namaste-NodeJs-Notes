 # 📦 JavaScript Modules (Comparison)

| CommonJS Modules (CJS) | ES Modules (MJS) |
|------------------------|------------------|
| **module.exports**     | **export**       |
| **require()**          | **import**       |
| Used by default in **Node.js** | Used by default in **React, Angular** |
| Synchronous loading    | Asynchronous loading |
| Older module system    | Modern JS standard |
| Code Runs in Non Strict Mode | Code Runs in Strict Mode |

---

For example in strict mode you cannot define variables without var, let or const you have to define something before the variable name
